# Installation 
To use the Xarm you need to install the Python SDK
https://github.com/xArm-Developer/xArm-Python-SDK

* git clone git@github.com:xArm-Developer/xArm-Python-SDK.git
* cd xArm-Python-SDK
* python setup.py install
